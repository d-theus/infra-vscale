# vscale-infra

Inspired by [terraform](https://www.terraform.io/).

## Usage

```
$> bundle
$> VSCALE_TOKEN=xyz bundle exec bin/info
$> VSCALE_TOKEN=xyz bundle exec bin/plan state.yml
$> VSCALE_TOKEN=xyz bundle exec bin/apply state.yml

```
